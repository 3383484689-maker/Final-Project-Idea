# AI Portrait Stylization — Final Project (All-in-One Creation)

## Short Idea Description (for final dashboard)
An AI-powered web application that transforms personal daily photos into artistic portrait styles such as cinematic, studio, film, and magazine looks. Built with Streamlit and using Replicate (Stable Diffusion img2img) or OpenAI (DALL·E / Image Edit) as the image generation backend.
